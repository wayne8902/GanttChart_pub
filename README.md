# 甘特圖專案管理

離線 Windows 桌面程式，供專案與任務進度管理，並以甘特圖檢視時程。

## 下載

請至 [Releases](../../releases) 下載最新版本的 `GanttChart-v*.zip`。

## 安裝與執行

1. 下載 zip 並解壓縮
2. 進入 `GanttChart-v*` 資料夾
3. 雙擊 **`launcher.exe`** 啟動

**無需安裝 Python，無需連網即可使用。**

## 系統需求

- Windows 10 或以上

## 基本使用

- **專案**：新增、編輯、刪除、下拉切換
- **任務**：新增、編輯、任務編輯器、進度 0～100%、長條顏色自訂（8 色）、實際完成日與延期延伸顯示
- **甘特圖**：日／週／月刻度、橫向捲動、點擊長條編輯任務
- **匯出甘特圖**：以 PNG 儲存完整圖表（全部任務列與完整時間軸，與目前刻度一致）
- **設定**：淺色／深色外觀、使用指南、關於（含手動檢查更新）
- **版本檢查**：`launcher.exe` 啟動前比對 GitHub Release（失敗靜默）；有新版本時 MessageBox 提示，仍照常開啟主程式
- **資料本機儲存**，按「儲存變更」寫入檔案

## 資料存放位置

程式資料存放在 `launcher.exe` 同層的 `gantt_sys` 資料夾：

| 檔案 | 說明 |
|------|------|
| `projects.json` | 專案與任務資料 |
| `ui-settings.json` | 介面設定（如淺色／深色） |
| `about.toml` | 關於資訊 |
| `guide.toml` | 使用指南 |

解壓後已預帶範例專案；若缺少檔案，程式會自動建立預設內容。請定期備份 `gantt_sys` 資料夾。

## 注意事項

- 請執行 **`launcher.exe`**，不要到 `core` 資料夾執行 `GanttChart.exe`
- 請勿只複製 `launcher.exe`，需保留整個 `GanttChart-v*` 資料夾（含 `core` 子資料夾）
- 更新版本時，可先備份舊版 `gantt_sys` 再覆蓋程式檔
- 本程式為離線使用，資料僅存於本機
